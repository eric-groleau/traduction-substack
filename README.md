# traduction-substack
Système de traduction d'une série d'articles sur substack
**traduction-substack** est un projet visant à automatiser la traduction d’articles publiés sur Substack vers différentes langues. Le système télécharge les articles et leurs images, extrait et traduit le texte, et analyse les images pour détecter la présence de texte (aucun texte, texte seul, ou image mixte). Selon la catégorie, les images sont soit conservées, soit remplacées par leur contenu textuel traduit, soit accompagnées d’une légende traduite. Le projet génère également un PDF traduit automatiquement afin de servir de base de comparaison, même si l’objectif principal est de produire un document éditable (ODT) permettant des corrections humaines. L’architecture est conçue pour être générique : elle peut s’appliquer à différentes séries Substack et à différentes langues de traduction.
